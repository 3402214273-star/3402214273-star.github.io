支出明细精简版

在 veeto.cn 支出明细页面自动过滤，只显示你需要的内容

手机号短信内容成功数量时间
1
安装 Tampermonkey
Chrome/Edge 商店搜索 Tampermonkey 并安装
2
安装脚本
点击下方按钮安装
3
刷新页面
登录 veeto.cn -> 支出明细页面 -> 右上角出现切换按钮
点击安装脚本 查看演示效果
1
复制以下代码
点击下方“复制代码”按钮
2
新建书签
浏览器菜单 -> 书签 -> 新建 -> 名称写“支出明细”，网址随便填 -> 保存
3
编辑书签
长按新书签 -> 编辑 -> 清空网址 -> 粘贴代码 -> 保存
4
使用
登录 veeto.cn -> 支出明细页面 -> 点这个书签
(function(){var c=["成功数量","手机号","短信内容","时间"];var t=document.querySelector("table");if(!t){alert("未找到表格");return}var idx=[];t.querySelectorAll("th").forEach(function(th,i){for(var k of c){if(th.textContent.trim().includes(k)){idx.push(i);break}}});if(idx.length<2){alert("未找到对应列");return}t.querySelectorAll("tr").forEach(function(r){r.querySelectorAll("td,th").forEach(function(cell,i){cell.style.display=idx.includes(i)?"":"none"})})})()
